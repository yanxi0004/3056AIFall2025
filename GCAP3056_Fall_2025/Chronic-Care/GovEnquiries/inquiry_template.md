# Government Inquiry Template

## Instructions for Updates
- Use this template for new government inquiries
- Copy and rename for each specific inquiry
- Include all relevant contact information
- Document responses and follow-up actions

## Template Structure
```markdown
# Inquiry to: [Department/Bureau Name]
## Date: [YYYY-MM-DD]
## Subject: [Brief subject line]

## Inquiry Details
**Purpose**: [What you're asking for]
**Background**: [Context and relevance]
**Specific Questions**:
1. [Question 1]
2. [Question 2]
3. [Question 3]

## Contact Information
- **Department**: [Name]
- **Contact Person**: [Name and title]
- **Email**: [Email address]
- **Phone**: [Phone number]

## Follow-up Actions
- [ ] [Action 1]
- [ ] [Action 2]

## Response Received
**Date**: [YYYY-MM-DD]
**Summary**: [Brief summary of response]
**Next Steps**: [What to do with the response]
```
