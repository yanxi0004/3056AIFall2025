# HKO Chatbot Project - Meeting Notes

## Instructions for Updates
- Add meeting dates and participants
- Document key discussion points
- Record decisions and action items
- Include follow-up tasks and deadlines

## Template Structure
```markdown
# Meeting Date: [YYYY-MM-DD]
## Participants
- [Name 1]
- [Name 2]

## Agenda Items
1. [Item 1]
2. [Item 2]

## Discussion Points
- [Key point 1]
- [Key point 2]

## Decisions Made
- [Decision 1]
- [Decision 2]

## Action Items
- [ ] [Task 1] - [Assigned to] - [Due date]
- [ ] [Task 2] - [Assigned to] - [Due date]

## Next Meeting
- Date: [YYYY-MM-DD]
- Agenda: [Brief description]
```
