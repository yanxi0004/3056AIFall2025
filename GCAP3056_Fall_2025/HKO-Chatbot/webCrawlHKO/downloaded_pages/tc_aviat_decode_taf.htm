剖析機場天氣預報(TAF)電碼的意義

TAF傳達以下氣象資料：

發出預報的機場名稱、TAF有效期、風向∕風速∕陣風、能見度、天氣、雲層、最高及最底溫度、天氣要素的顯著轉變。

能見度： 9999 = 10公里或以上、7000 = 7公里、0000 = 50米以下
CAVOK = 天氣及能見度良好
天氣： '-' = 輕微、 '+' = 大、VC = 附近、MI = 淺薄、BC = 幾陣、
SH = 驟雨、PR = 部分、TS = 雷暴、DZ = 毛毛雨、RA = 雨、
BR = 薄霧、FG = 霧、HZ = 煙霞
+SHRA = 大驟雨
-DZ = 輕微毛毛雨
雲層： 把天空分為八份，以表達雲層遮蓋天空的比例
FEW = 一至兩份雲、SCT = 三至四份雲、BKN =五至七份雲、OVC = 八份雲、
SKC = 天朗氣清

顯著轉變：BECMG = 逐漸轉變、TEMPO = 間中

根據區域航空協議，當機場天氣預報包括氣溫預報時，則應提供預報有效期內的最高氣溫及最低氣溫預測，並且註明這些氣溫出現的時間。 若預報有效期內有兩個相同的最高氣溫或最低氣溫預測，只會採用較早出現的那一個。

若情況符合某些準則，有關機場可以發出修訂預報，並在前面加上關鍵字眼 'TAF AMD'，以資識別。

例子：

TAF VHHH 160500Z 1606/1712 13018KT 9000 BKN020 TX30/1606Z TN26/1622Z BECMG 1606/1608 SCT015CB BKN020 TEMPO 1608/1612 17025G40KT 1000 TSRA SCT010CB BKN020 FM161200 15015KT 9999 NSW BKN020 BKN100=

應解碼為：

電碼 意義
TAF VHHH TAF由香港國際機場發出
160500Z 日期為該月份的十六日，時間為0500 UTC(協調世界時)
1606/1712 有效期由0600 UTC至翌日1200 UTC
13018KT 地面風向為130度(東南偏東風)、風速18海里/小時
9000 能見度9公里
BKN020 2000呎有疏鬆雲層
TX30/1606Z 最高氣溫為攝氏30度，約在十六日， 0600 UTC出現
TN26/1622Z 最低氣溫為攝氏26度，約在十六日， 2200 UTC出現
BECMG 1606/1608 SCT015CB BKN020 在0600 UTC至0800 UTC之間雲層逐漸轉變，轉變後1500呎有零散雲層，包括積雨雲，2000呎有疏鬆雲層
TEMPO 1608/1612 17025G40KT 1000 TSRA SCT010CB BKN020 在0800 UTC至1200 UTC 之間，地面風向間中為170度(東南偏南風)，風速25海里/小時，陣風達40海里/小時；能見度在雷暴及和緩雨勢中為1000米；1000呎有零散雲層，包括積雨雲，2000呎有疏鬆雲層
FM161200 15015KT 9999 NSW BKN020 BKN100 由1200UTC開始，地面風向為150度(東南偏南風)，風速15海里/小時；能見度10公里或以上；並無顯著天氣狀況；2000呎有疏鬆雲層，10000呎亦有疏鬆雲層
